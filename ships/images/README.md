# Ship Images

This folder contains background images for ship cards in the Star Citizen Hangar Manager extension.

## Image Naming Convention

Images should be named using the following format:
- **Lowercase only**
- **Replace spaces with dashes (-)**
- **Remove special characters**
- **Include manufacturer if part of common name**

### Examples

| Ship Name | File Name |
|-----------|-----------|
| Aegis Avenger Titan | `aegis-avenger-titan.jpg` |
| RSI Aurora MR | `rsi-aurora-mr.jpg` |
| Anvil C8X Pisces Expedition | `anvil-c8x-pisces-expedition.jpg` |
| Origin 890 Jump | `origin-890-jump.jpg` |
| MISC Freelancer MAX | `misc-freelancer-max.jpg` |

## Image Specifications

- **Format**: JPG or PNG
- **Aspect Ratio**: 16:9 recommended
- **Minimum Resolution**: 800x450px
- **Maximum File Size**: 500KB (optimize for web)
- **Quality**: High-quality official artwork or screenshots

## Contributing Images

1. Find or create high-quality ship images
2. Name according to convention above
3. Optimize for web (compress without losing quality)
4. Submit via pull request

## Image Sources

Recommended sources for ship images:
- Official Star Citizen website
- RSI Ship Matrix
- Official Star Citizen promotional materials
- High-quality in-game screenshots (no HUD)
